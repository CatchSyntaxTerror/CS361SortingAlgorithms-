## Benchmarker.jar
Arguments:
`-generate <start> <end>`: Generate data files of size 2^`<start>` to 2^`<end>`
`-benchmark <start> <end> <trials>`: Use generated data files to benchmark in 
range for `<trials>`

Both assume data is stored in a "data" folder in the working directory.
